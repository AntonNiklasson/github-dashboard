## What I do

I help manage a project todo list stored in TODO.md. I can:
- Read and display current tasks
- Add new tasks to the list
- Mark tasks as complete
- Suggest improvements to task descriptions

## How to use me

When the user wants to manage tasks:
1. Read TODO.md to see current tasks
2. Add new tasks in the format "- [ ] Description" under appropriate section
3. Mark tasks complete by changing "- [ ]" to "- [x]"
4. Show the user the current state after changes

## Task format

Tasks should use checkmark format:
- Unchecked: `- [ ]`
- Checked: `- [x]`

Organize by sections: Features, Bug Fixes, Tech Debt, Ideas
